# monitor-shutdown-WEB-2.0
用于媒体服务器的轻量化简易面板


可以在windows系统上创建一个web面板，可以查看CPU占用，内存占用，网络上行及其下行量，以及发送重启指令和关机指令


使用需要psutil以及flask获取系统资源占用情况


可使用pip命令安装


pip install flask psutil
